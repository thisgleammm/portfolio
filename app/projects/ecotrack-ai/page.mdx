    ## Overview
    
    EcoTrack AI is a flagship project developed to solve real-world industrial waste classification issues using **Computer Vision** and **Deep Learning**.
    I built this system to identify 8 distinct waste categories with a high precision rate of 94%.

    ## Key Features
    
    - **Real-time Detection**: Edge-optimized inference using ONNX.
    - **Visual Insights**: Interactive dashboard built with Next.js and Recharts.
    - **Automated Alerts**: Integration with Telegram Bot for instant reporting.
    
    ## Tech Stack
    
    ```python

    def classify_waste(image_tensor):
        prediction = model.predict(image_tensor)
        return categories[prediction.argmax()]
    ```
    
    We also used:
    - **Next.js 15** for the frontend.
    - **Rust** for performance-critical tasks.
    - **Docker** for containerized deployment.

    ## Challenges
    
    Handling data scarcity for specific waste types was the biggest hurdle. 
    I implemented **Data Augmentation** strategies and **Synthetic Data Generation** to balance the dataset.
